## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)
- Code editor (VS Code, Sublime Text, etc.)

### Installation & Setup

1. **Download or Clone** the repository:

   ```bash
   git clone https://github.com/yourusername/choicy.git
   ```

2. **Navigate** to the project folder:

   ```bash
   cd choicy
   ```

3. **Open** `index.html` in your browser or set up a local server:

   ```bash
   # Using Python 3
   python -m http.server 8000

   # Using PHP
   php -S localhost:8000

   # Using Node.js (http-server)
   npx http-server
   ```

4. **Visit** `http://localhost:8000` in your browser

### ğŸ¨ Customization

#### Changing Colors

Edit the SCSS variables in `assets/scss/abstracts/_variables.scss`:

```scss
$primary-color: #your-color;
$secondary-color: #your-color;
```

Then compile SCSS to CSS:

```bash
sass assets/scss/main.scss assets/css/main.css
```

#### Modifying Content

Simply open any HTML file and update the text, images, and links to match your needs.

#### Adding New Pages

Use any existing page as a template and customize it according to your requirements.
